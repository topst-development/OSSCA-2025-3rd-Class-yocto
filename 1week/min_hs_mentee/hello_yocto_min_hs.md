git init (일반 저장소)
로컬 개발용으로써 직접 코드를 작성하고 수정
git add, commit 가능

git init --bare (Bare 저장소)
중앙 저장소용으로 여러 개발자가 push/pull하는 공유 저장소
클라우드 저장소와 같은 역할

stash 명령어
stash는 로컬에만 저장되며 다른 개발자와 공유되지 않음
중요한 코드는 커밋 권장하고 stash는 임시 저장용
