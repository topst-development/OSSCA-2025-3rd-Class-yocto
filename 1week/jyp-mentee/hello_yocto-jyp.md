#### git init 과  git init --bare 차이
+ git init (일반 저장소)
  로컬 개발용. 여기서 코드를 편집하고 커밋을 만든다.
  실제 파일을 수정/빌드/실행할 수 있음.
+ git init --bare
  주로 중앙 원격 저장소(서버용)로 사용.
  개발자는 이 저장소에 push/pull만 하고, 직접 파일 편집은 하지 않음.
  코드 파일 없이 Git 데이터만 존재.
